# Chuck's Programming Toolbox

A personal collection of programming knowledge and techniques, primarily focused on LeetCode-style problem-solving. This toolbox serves as a reference and learning resource for algorithms, data structures, and common patterns I've encountered and studied.

## Overview

This toolbox contains implementations, explanations, and insights for various programming concepts organized in a card-based structure. Each card represents a specific algorithm, pattern, or technique with my personal notes and explanations.

## Structure

Each concept is presented as a **card** containing:
- **Implementation**: Code examples in TypeScript/JavaScript
- **Explanation**: My personal notes and understanding
- **Time/Space Complexity**: Analysis when applicable
- **Use Cases**: When to apply the technique
- **Related Problems**: Links to similar LeetCode problems

## Organization

Cards are organized by:
- **Classification**: The category (sorts, searches, algos, etc.)
- **Difficulty**: Complexity level when applicable
- **Tags**: For cross-referencing related concepts
